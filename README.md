# WebRTC 
- if `A` wants to connect to -> `B`.
- `A` creates an **offer** and generates SDP, the offer is basically the SDP.
- `A` signals the offer shoehow to `B`
- `B` creates the **answer** after setting `A's` offer
- connection created!!
